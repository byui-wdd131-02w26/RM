Restaurant Menu Site Plan

Rugen Myrto

WDD 131-2

Overview

Purpose

To provide an online menu for customers to browse and decide on their orders, without needing a physical menu.

Audience

Customers who want to view my dad's restaurant's menu.

Dynamic elements

JavaScript is used to make the menu interactive. The menu items are stored in an array of objects and displayed dynamically using functions. Users can filter items by category and search in real time. This is done using event listeners and the filter() method. Conditional statements (if/else) decide which items are shown. There is also a language toggle (English/Albanian) that updates the page content by modifying the DOM. The site uses array methods like forEach() to display items and reduce() to calculate the average price, demonstrating organized and dynamic code.

Content

Home page

The home page introduces City Pizza and shows a digital menu for customers. It includes a short description of the restaurant and its food. The menu is organized into categories like pizza, sandwiches, salads, antipasta, and drinks. Users can filter items and search for specific foods. Each item shows its name, description, and price. The page also includes an image of pizza to make it more visually appealing, along with basic information like opening hours.
